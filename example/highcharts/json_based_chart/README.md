# Chart generated from JSON

**Highcharts demo**
Line chart created directly from a JSON configuration using
HighchartsJsonOptions.fromJson. This example visualizes U.S. solar employment
growth by job category from 2010 to 2022, based on data from the IREC Solar
Jobs Census.

# How to run

In the demo directory:

```
flutter create .
flutter pub get
flutter run
```
