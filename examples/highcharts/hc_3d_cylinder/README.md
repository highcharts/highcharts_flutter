# 3D cylinder demo

**Highcharts demo**
Chart showing basic use of 3D cylindrical columns. A 3D cylinder chart
        is similar to a 3D column chart, with a different shape. The chart
        visualizes confirmed COVID-19 cases for women in Norway.
